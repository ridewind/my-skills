### C-001: SQL Injection Vulnerability

**Found by**: security-analyzer

**File**: `src/auth/login.py`
**Lines**: 45-52

**Description**:
Direct string formatting in SQL query creates SQL injection vulnerability.

**Current Code**:
```python
query = f"SELECT * FROM users WHERE username = '{username}'"
cursor.execute(query)
```

**Recommended Fix**:
```python

```

**Impact**: Attackers can execute arbitrary SQL commands, potentially accessing or modifying all database data.
